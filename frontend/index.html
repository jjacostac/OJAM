    Agregar tu injusticia

    Tu reporte entra a una cola de revisión. Una vez aprobado por el equipo de curaduría, aparecerá publicado en el mapa. La fecha y hora de carga se registran automáticamente al enviar.

    Ubicación del problema *

    Usa el GPS del dispositivo o toca el minimapa para marcar el punto. Puedes arrastrar el pin o corregir las coordenadas a mano si el GPS falla.

    Si reportas de forma anónima no se guarda ningún dato de contacto y el equipo de curaduría no podrá escribirte para pedir más información sobre el caso. Si dejas un contacto, se usa únicamente para eso: nunca se publica en el mapa ni en los datos abiertos, y solo lo ve el equipo administrador.
